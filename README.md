# 📊 Customer Retention & Churn Analysis

## 🔍 Project Overview
This project analyzes customer churn to understand why customers leave and how businesses can improve retention.

## 📁 Dataset
Telco Customer Churn Dataset

## 🛠 Tools Used
- Power BI
- Excel

## 📊 Dashboard Features
- KPI Cards (Total Customers, Churn Customers, Churn Rate)
- Churn Distribution
- Churn by Contract Type
- Customer Tenure Analysis
- Impact of Monthly Charges on Churn
- Churn by Payment Method

## 📈 Key Insights
- Customers with month-to-month contracts have the highest churn
- Higher monthly charges lead to increased churn
- Customers with shorter tenure are more likely to leave
- Electronic check payment users show higher churn

## 💡 Recommendations
- Promote long-term contracts
- Improve onboarding experience
- Offer loyalty benefits to retain customers
